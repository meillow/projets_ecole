Le Memory

En 2018, on nous a demandé de réaliser le jeux du Memory en C. Le jeu du Memory consiste à choisir deux cartes de dos et de voir leur face est similaire. Si c'est le cas alors on marque un point. Sinon on doit repiocher. Un chronomètre, un mode pause et un mode triche nous étaient aussi demandés.

Pour les projets en C, il nous est demandé d'utiliser la bibliothèque graphique de l'iut, ce qui implique une difficulté supplémentaire puisque cette bibliothèque est limitée.

Ce projet est mon premier projet d'informatique en groupe. J'ai eu quelques difficultés pour l'appréhender. Mais grâce à l'aide de mon binôme, nous avons pu le réaliser entièrement dans les temps.
